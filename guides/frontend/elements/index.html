Frontend Docs: Elements

To add one of these elements to your web-app, import it like this:

import "@holochain-open-dev/profiles/agent-avatar";

Keep in mind that these elements should be placed inside a <profiles-context> that provides an initialized ProfilesStore. Follow the Adding the UI guide to get started, or read this to know more about the context pattern.

Click on the selector on the right to look at all the elements available.
